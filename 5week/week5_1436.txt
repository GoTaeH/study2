1. 문제 해결 기법
    - 종말을 나타내는 수인 666(6이 적어도 3개 이상 연속으로 들어가는 수)을 사용하여 영화제목을 만드려고 한다. 
    영화는 시리즈물이며 정수 N을 입력받아 N번째 영화 제목에 들어갈 수를 구하는 문제이다. 이 문제는 차례대로 규칙에 맞게 
    수행해야 하므로, 브루트포스 방식을 사용하여 해결해야 한다.

2. 논리 구현
버퍼리더 br을 선언한다.
타입이 int인 N을 선언하여 br.readLine()으로 입력 받는다.    // N번째 영화

타입이 int인 endNum을 666으로 정의한다.     // 종말을 나타내는 수 666
타입이 int인 cnt를 1로 정의한다.    // count

while(cnt != N) {   // count가 N이 아닌 동안
    endNum을 증가시킨다.    // endNum++
    if(666이 포함되는지 문자열을 검사한다.)     //String.valueOf(endNum).contains("666")
        cnt를 증가시킨다.   //cnt++
}
endNum을 출력한다.
