1. 문제 해결 기법
    - ccw 알고리즘을 구현하는 문제이다. 점 3개 P1(x1, y1), P2, P3을 입력 받아 점 3개를 이어서
    방향(반시계, 시계, 일직선)을 확인한다. 따라서 ccw 공식 (x1y2+x2y3+x3y1)-(y1x2+y2x3+y3x1)을 이용하여 문제를 해결한다.

2. 논리 구현
Scanner를 선언한다.
int x1, y1, x2, y2, x3, y3을 정의하고, 입력받는다. (sc.nextInt())
ccw 공식을 사용하여 입력받은 점을 계산하는 식을 정의한다. (int ccw = (x1 * y2 + x2 * y3 + x3 * y1) - (y1 * x2 + y2 * x3 + y3 * x1))

if ccw > 0 이면 (반시계 방향)
    1을 출력한다.
else if ccw < 0 이면 (시계 방향)
    -1을 출력한다.
else (일직선)
    0을 출력한다.
